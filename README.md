# 將棋（Shogi Plus）

本專案為可離線執行的將棋對戰遊戲，支援標準走子、升變與打入規則，提供本機雙人對戰與日式風格 UI。

## 特色
- 本機雙人對戰
- 連線雙人對戰（房間碼，WebRTC）
- 合法走子與升變判定
- 打入規則（含二步、打步詰）
- 王手、將死與千日手和局
- 日式風格棋盤與介面

## 目前未涵蓋
- 入玉裁定與投了
- AI 對戰與難度分級
- 走譜、悔棋、存檔/讀檔

## 使用方式
1. 直接用瀏覽器開啟 `index.html`
2. 或使用任意靜態伺服器啟動後開啟

## 連線對戰啟動
1. 安裝依賴：`npm install`
2. 啟動信令伺服器：`npm start`
3. 以瀏覽器開啟 `index.html`，輸入房間碼連線

預設信令伺服器使用 `ws://localhost:8080`，如用 HTTPS 部署會自動改用 `wss`。

## 操作說明
- 點擊棋子可顯示可走位置
- 再點擊目的格即可移動
- 進入升變區會詢問是否升變（必須升變時自動升變）
- 點擊持駒可在空格打入

## 專案結構
- `index.html`：畫面結構
- `style.css`：日式風格樣式
- `app.js`：棋盤邏輯與規則
- `WORK_PLAN.md`：工作計畫與待辦清單
